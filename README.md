# ✈️ Smart Airport System — Network Design
 
A 7-floor airport network simulation built in **Cisco Packet Tracer 8.2**.
 
---
 
## 🔑 Key Features
 
- 3-Tier Hierarchical Model (Core, Distribution, Access)
- 7 VLANs — one per floor (192.168.10.0 to 192.168.70.0)
- Dual Routers & Multilayer Switches for redundancy
- Centralized Server Farm (Web, DNS, FTP, Email)
- IoT wind sensor on ATC floor (Floor 7) for safety alerts
